# UI Components

`@promethean/ui-components` provides design tokens and reusable Web Components.

## Design Tokens
Tokens are applied as CSS variables using `applyDesignTokens()`.

## Components
- `<ui-file-explorer>` – basic container for file navigation UIs.
- `<ui-chat-panel>` – wrapper element for chat content.

Call `registerUiComponents()` to define the custom elements.
