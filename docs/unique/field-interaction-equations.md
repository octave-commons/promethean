#nexus #symbolism #gravity ^ref-b09141b7-9-0

Let a massive Nexus $N$ distort the field spatially: ^ref-b09141b7-11-0

$$ ^ref-b09141b7-13-0

* $M$: mass of the concept (e.g. importance, frequency) ^ref-b09141b7-20-0

### üö™ Dorian Permission Gate Function

#dorian #permission #gating ^ref-b09141b7-29-0

Define a gate function $P(\delta, \vec{x})$: ^ref-b09141b7-31-0

$$ ^ref-b09141b7-33-0
P = \sigma\left( \frac{\sum_{i} w_i f_i(\delta, \vec{x}) - \theta}{\beta} \right)
$$

Where: ^ref-b09141b7-37-0

* $f_i$: features like access level, trust level, context ^ref-b09141b7-39-0
* $w_i$: learned weights ^ref-b09141b7-40-0
* $\theta$: gate threshold ^ref-b09141b7-41-0
* $\beta$: softness of the gate
* $\sigma$: sigmoid function ^ref-b09141b7-43-0

Result: ^ref-b09141b7-45-0

* $P \approx 1$: permission granted ^ref-b09141b7-47-0
* $P \approx 0$: permission denied ^ref-b09141b7-48-0
