### Core Interfaces to Implement

#### 1. ScarContext Interface

```typescript
interface ScarContext {
  reason: string;
  eventLog: EventLogEntry[];
  previousScars: ScarRecord[];
  searchResults: SearchResult[];
  metadata: {
    tag: string;
    narrative: string;
  };
  llmOperations: LLMOperation[];
  gitHistory: GitCommit[];
}
```

#### 2. EventLogEntry Interface

```typescript
interface EventLogEntry {
  timestamp: Date;
  operation: string;
  details: Record<string, any>;
  severity: 'info' | 'warning' | 'error';
}
```

#### 3. ScarRecord Interface

```typescript
interface ScarRecord {
  start: string; // Git SHA
  end: string; // Git SHA
  tag: string;
  story: string;
  timestamp: Date;
}
```

#### 4. SearchResult Interface

```typescript
interface SearchResult {
  taskId: string;
  title: string;
  relevance: number;
  snippet: string;
}
```

#### 5. LLMOperation Interface

```typescript
interface LLMOperation {
  id: string;
  operation: string;
  input: any;
  output: any;
  timestamp: Date;
  tokensUsed?: number;
}
```

#### 6. GitCommit Interface

```typescript
interface GitCommit {
  sha: string;
  message: string;
  author: string;
  timestamp: Date;
  files: string[];
}
```
