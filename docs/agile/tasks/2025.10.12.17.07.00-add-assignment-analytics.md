### API Design
```bash
# Analytics commands
pnpm kanban analytics performance --agent <agent-id> --days 30
pnpm kanban analytics patterns --type assignments --trend
pnpm kanban analytics optimization --recommendations
pnpm kanban analytics dashboard --refresh

# Reporting
pnpm kanban report generate --type performance --format pdf
pnpm kanban report schedule --type weekly --recipients team@example.com
pnpm kanban report custom --metrics "completion_time,success_rate"

# Metrics and KPIs
pnpm kanban metrics list
pnpm kanban metrics track --name "custom_metric" --formula "..."
pnpm kanban kpi dashboard --team "development"
```
