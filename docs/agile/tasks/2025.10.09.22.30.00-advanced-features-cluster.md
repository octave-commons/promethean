## üîó Feature Specifications

### Enhanced CLI Commands

```bash
# CRUD Operations
kanban create --title "Task Title" --priority P1 --template bug-fix
kanban read <uuid> [--format json|markdown|table]
kanban update <uuid> --status in-progress --estimate 3
kanban delete <uuid> [--confirm]

# Advanced Search
kanban search --query "typescript" --status incoming --priority P1,P2
kanban filter --label bug --assigned-to user@example.com
kanban list --column done --completed-after 2025-10-01

# Bulk Operations
kanban bulk-update --filter "status:incoming" --set-status accepted
kanban bulk-assign --filter "priority:P1" --assign-to team-lead
kanban bulk-estimate --filter "status:ready" --auto-estimate
```

### AI-Powered Features

```bash
# AI Assistance
kanban ai suggest --based-on current-sprint
kanban ai breakdown --task-uuid <uuid>
kanban ai estimate --task-uuid <uuid> --complexity medium
kanban ai assign --task-uuid <uuid> --consider-skills true

# Natural Language
kanban create-natural "Fix the TypeScript build errors in cephalon package"
kanban find "tasks related to authentication issues"
kanban summarize --column in-progress
```

### MCP Integration

```bash
# MCP Server Management
kanban mcp start --port 8080
kanban mcp status
kanban mcp sync --with-remote
kanban mcp template --list
kanban mcp intent --process "create bug fix task"
```
