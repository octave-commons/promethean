# Evaluate optional JS services for TypeScript migration

Assess which optional services should be migrated to TypeScript and which can be deprecated.

## Services to Evaluate

- [ ] event-gateway
- [ ] event-hub
- [ ] health
- [ ] proxy

## Evaluation Criteria

- Current usage and dependencies
- Migration complexity vs benefit
- Alternative solutions available
- Maintenance overhead

## Acceptance Criteria

- [ ] Each service evaluated against criteria
- [ ] Decision documented for each service
- [ ] Migration plan created for services to migrate
- [ ] Deprecation plan created for services to drop
