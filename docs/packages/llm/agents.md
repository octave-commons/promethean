# Llm Service

## Overview

Large Language Model service that provides text generation capabilities with support for context, formatting, and tool integration. Offers HTTP and WebSocket interfaces for prompt processing, includes retry logic for reliability, and supports broker-based message publishing for asynchronous responses. Configurable to work with different LLM drivers and provides extensible tool calling functionality.
