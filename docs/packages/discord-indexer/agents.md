# Discord Indexer Service

## Overview

Discord indexer service that crawls and indexes Discord server content for search and analysis. Processes messages, channels, threads, and user information to create searchable indexes. Supports full-text search, metadata extraction, and content categorization. Integrates with vector databases for semantic search and provides APIs for content retrieval and analytics.
