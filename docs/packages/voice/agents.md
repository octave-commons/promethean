# Voice Service

## Overview

Discord voice service that manages voice channel operations including joining/leaving channels, voice recording, transcription, and text-to-speech playback. Provides RESTful endpoints for voice session management, speaker handling, and real-time audio processing. Integrates with Discord.js for voice connectivity and supports multiple concurrent users with individual recording and transcription capabilities.
