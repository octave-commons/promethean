> ## ðŸš€ Installing BRAT
> 
> 1. **Open Obsidian** and navigate to `Settings > Community Plugins`.
> 2. **Enable** community plugins if you haven't already.
> 3. Click on **"Browse"**, search for **"BRAT"**, and install the plugin named **"Obsidian42 - BRAT"**.
> 4. After installation, **enable** the BRAT plugin.
> 
> ---
> 
> ## ðŸ§ª Adding a Beta Plugin
> 
> 1. Obtain the GitHub repository URL of the plugin you wish to test (e.g., `https://github.com/username/plugin-name`).
> 2. Press `Ctrl+P` (or `Cmd+P` on macOS) to open the command palette.
> 3. Run the command **"BRAT: Add a beta plugin for testing"**.
> 4. Paste the GitHub URL into the prompt and confirm.
> 5. BRAT will download and install the plugin.
> 6. Go to `Settings > Community Plugins`, locate the newly installed plugin, and **enable** it.
> 
> ---
> 
> ## ðŸ”„ Updating Beta Plugins
> 
> - To update all beta plugins, open the command palette and run **"BRAT: Check for updates to all beta plugins and UPDATE"**.
> - Alternatively, enable auto-updates by going to `Settings > Obsidian42 - BRAT` and toggling the auto-update option.
> 
> ---
> 
> ## ðŸ§¹ Removing a Beta Plugin
> 
> 1. Navigate to `Settings > Obsidian42 - BRAT`.
> 2. Click the **"X"** next to the plugin you wish to remove from BRAT's management.
> 3. To completely uninstall the plugin, go to `Settings > Community Plugins`, find the plugin, and click **"Uninstall"**.
