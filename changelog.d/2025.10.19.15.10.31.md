# Tool Name Updates - Dot to Underscore Conversion

## Changed
- Updated all tool names in `@packages/opencode-client/src/plugins/` to use underscores instead of dots
- This change affects 12 plugin files with a total of 58 tool name updates

## Reason
Tool names with dots can cause issues in various contexts and underscores are more compatible across different systems and APIs. All tool functionality remains exactly the same, only the naming convention has changed.
