# Unified Agent Management API

## Added

- **UnifiedAgentManager** - High-level API for complete agent lifecycle management
- **createAgentSession()** - Single command to create session, assign task, and send initial message
- **AgentSession** interface - Unified session representation with task and status
- **Event-driven callbacks** - onStatusChange and onMessage handlers
- **CLI interface** - Complete command-line tool for agent management
- **Session pooling and batch processing** capabilities
- **Statistics and monitoring** built-in
- **Automatic cleanup** for old sessions

## Features

- Create agent sessions with task assignment in one operation
- Event-driven architecture with callbacks
- Session lifecycle management (create → start → interact → stop → close)
- Batch processing capabilities
- Real-time statistics and monitoring
- Interactive CLI mode
- Comprehensive error handling
- TypeScript support with full type definitions

## CLI Commands

- `agent-cli create <task>` - Create new session
- `agent-cli start <id>` - Start session
- `agent-cli send <id> <msg>` - Send message
- `agent-cli list` - List sessions
- `agent-cli interactive` - Interactive mode
- `agent-cli stats` - Show statistics
- `agent-cli cleanup` - Cleanup old sessions

## API Examples

```typescript
// Single command session creation
const session = await createAgentSession(
  'Analyze performance',
  'Focus on database queries',
  { priority: 'high' },
  { autoStart: true },
);

// Event-driven handling
await createAgentSession(
  'Task',
  undefined,
  {},
  {
    onStatusChange: (id, old, newStatus) => console.log(`${id}: ${old} → ${newStatus}`),
    onMessage: (id, msg) => console.log(`Message: ${msg.content}`),
  },
);
```
