# Fixed clojure-mcp Configuration

## Problem

The clojure-mcp agents in `.clojure-mcp/config.edn` had configuration issues with non-existent tool references that prevented agents from loading properly.

## Root Cause Analysis

- Initial configuration referenced `:serena_*` tools that don't exist in clojure-mcp environment
- Some tools like `:clj-kondo-lint_clojure` were incorrectly named
- Tool `:clojure_edit_replace_sexp` does exist (confirmed via GitHub search)

## Solution

1. **Removed invalid tool references**: Cleaned up all `:serena_*` tool references
2. **Fixed tool naming**: Corrected `:clj-kondo-lint_clojure` to proper naming
3. **Restored valid tools**: Confirmed `:clojure_edit_replace_sexp` exists and restored references
4. **Validated configuration**: EDN syntax now parses successfully
