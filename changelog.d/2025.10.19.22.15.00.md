# CLI Consolidation - Unified Agent Management

## Changes Made

### Consolidated CLI Tools

- **Merged** `agent-cli` functionality into `opencode-client`
- **Removed** redundant `agent-cli` binary and source files
- **Added** unified `agent` command group to main CLI with all agent session management capabilities

### New CLI Structure

- `pnpm opencode` or `pnpm opencode-cli` - Single unified CLI
- `pnpm opencode agent` - All agent session management commands:
  - `create` - Create new agent sessions
  - `start` - Start existing sessions
  - `stop` - Stop sessions
  - `send` - Send messages to sessions
  - `close` - Close sessions
  - `list` - List active sessions
  - `info` - Get session details
  - `stats` - Show session statistics
  - `cleanup` - Clean up old sessions
  - `interactive` - Interactive session management mode

### Benefits

- **Eliminates confusion** between two overlapping CLI tools
- **Single entry point** for all OpenCode and agent operations
- **Preserves all functionality** from both original CLIs
- **Better user experience** with unified interface

### Backward Compatibility

- `pnpm opencode-cli` still works (alias to unified CLI)
- All existing commands and options preserved
- No breaking changes to command interfaces

## Files Modified

- `packages/opencode-client/src/cli.ts` - Added unified agent commands
- `packages/opencode-client/package.json` - Updated binaries, removed agent-cli
- `package.json` - Added opencode alias
- `packages/opencode-client/src/cli/AgentCLI.ts` - **REMOVED**
- `packages/opencode-client/dist/cli/AgentCLI.js` - **REMOVED**

## Testing
