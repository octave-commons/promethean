* Description
The =codex-hub= layer integrates Codex workflows into Spacemacs:
context-aware tasks, buffer fixing, and review flows powered by local/remote LLMs.
** Features:
  - Autocomplete
  - Lint
  - Refactor
  - ...

* Install
To use this configuration layer, add it to your =~/.spacemacs=. You will need to
add =codex-hub= to the existing =dotspacemacs-configuration-layers= list in this
file.

* Key bindings

| Key Binding  | Command             | Description          |
|--------------+---------------------+----------------------|
| ~SPC a x t~  | codex-hub-task      | Run task with context|
| ~SPC a x f~  | codex-hub-fix-buffer| Repair current buffer|
| ~SPC a x r~  | codex-hub-review    | Review changes       |
