# OpenCode Session Manager Frontend

Frontend package for the OpenCode session management interface.

## Development

```bash
# Build the frontend
pnpm --filter @promethean/opencode-session-manager-frontend build

# Watch for changes
pnpm --filter @promethean/opencode-session-manager-frontend watch:shadow

# Run tests
pnpm --filter @promethean/opencode-session-manager-frontend test
```
